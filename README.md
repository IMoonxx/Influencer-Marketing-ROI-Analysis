# Influencer-Marketing-ROI-Analysis
Excel and Power BI project analyzing influencer marketing ROI using correlation, regression, and performance metrics to optimize campaign decisions.

## Key Features
- ROI calculation for influencer campaigns
- Correlation analysis between engagement and revenue
- Regression analysis to assess impact of engagement
- Influencer segmentation (Micro, Mid, Macro)
- Interactive Power BI dashboard

## Key Insights
- Engagement showed a weak correlation with revenue (r = 0.034)
- Conversions had a stronger influence on revenue
- Mid size influencers delivered higher ROI compared to macro influencers

## Tools Used
- Excel
- Power BI
